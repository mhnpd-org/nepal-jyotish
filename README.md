## Introduction

- Its host the nepali jyotish software and blogs. It internally used @mhnpd-org/panchange packages for all the jytoish calculations. Its mainly hold the UI and UX part of the software.

- Deployed to cloudflare pages.
- Its closed source project. Everything is hosted in private repository.
