# राशि र तिनको विशेषता

राशि ज्योतिष शास्त्रको आधारभूत तत्व हो। राशि चक्रमा बाह्र राशिहरू छन् जसले व्यक्तिको विभिन्न गुणहरू प्रतिनिधित्व गर्छन्।

## राशि के हो?

राशि आकाशको एक भाग हो जसमा सूर्य, चन्द्र र अन्य ग्रहहरू भ्रमण गर्छन्। प्रत्येक राशि ३०° को हुन्छ।

### राशि चक्र
- **कुल राशि:** १२
- **प्रत्येक राशिको विस्तार:** ३०°
- **कुल:** ३६०° (पूर्ण चक्र)

## बाह्र राशिहरू

- **गुण:** चल

- **नीच:** शनि (२०°)

**स्वभाव:**

- नेतृत्व क्षमता
- आवेगी
- प्रतिस्पर्धी

- **स्वामी:** शुक्र
- **उच्च:** चन्द्र (३°)
- **नीच:** - 

**स्वभाव:**

## राशि वर्गीकरण

### तत्व अनुसार

**अग्नि राशि (उत्साही, सक्रिय):**
- मेष, सिंह, धनु

**पृथ्वी राशि (व्यावहारिक, स्थिर):**
- वृषभ, कन्या, मकर

**वायु राशि (बौद्धिक, सामाजिक):**
- मिथुन, तुला, कुम्भ

**जल राशि (भावुक, अन्तर्मुखी):**
- कर्क, वृश्चिक, मीन

### गुण अनुसार

**चल (Cardinal - नेतृत्व):**
- मेष, कर्क, तुला, मकर

**स्थिर (Fixed - दृढता):**
- वृषभ, सिंह, वृश्चिक, कुम्भ

**द्विस्वभाव (Mutable - लचिलो):**
- मिथुन, कन्या, धनु, मीन

## सूर्य राशि vs चन्द्र राशि

### सूर्य राशि
- जन्म मितिअनुसार
- बाहिरी व्यक्तित्व
- जीवन उद्देश्य
- पश्चिमी ज्योतिषमा प्रचलित

### चन्द्र राशि
- जन्म समयअनुसार
- आन्तरिक व्यक्तित्व
- भावनात्मक प्रकृति
- भारतीय ज्योतिषमा महत्वपूर्ण

## राशि र स्वास्थ्य

प्रत्येक राशिले शरीरको एक भाग प्रतिनिधित्व गर्छ:

| राशि | शरीर भाग |
|------|----------|
| मेष | टाउको, अनुहार |
| वृषभ | घाँटी, गला |
| मिथुन | हात, फोक्सो |
| कर्क | छाती, पेट |
| सिंह | मुटु, पीठ |
| कन्या | पाचन तन्त्र |
| तुला | मृगौला |
| वृश्चिक | प्रजनन अंग |
| धनु | जाँघ, कल्जो |
| मकर | घुँडा, हड्डी |
| कुम्भ | पिण्डली, रगत संचार |
| मीन | खुट्टा, लसिका तन्त्र |

## निष्कर्ष

राशि हाम्रो व्यक्तित्व र जीवनको एक महत्वपूर्ण पक्ष हो। तर, यो केवल एक पक्ष मात्र हो। सम्पूर्ण कुण्डली विश्लेषणले पूर्ण चित्र प्रदान गर्छ।

आफ्नो राशिको गुण र चुनौतीहरू बुझेर, हामी राम्रो जीवन जिउन सक्छौं।
